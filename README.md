# Simple Monopoly

## Project Description
A modern twist on the classic board game Monopoly, designed for an intuitive online experience.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Omer1405/simple-monopoly.git
   ```
2. Navigate into the project directory:
   ```bash
   cd simple-monopoly
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Gameplay Features
- Multiplayer support for up to 4 players
- Different board layouts for custom gameplay

## Multiplayer Setup
To host a multiplayer game, run the server using:
```bash
npm start
```
Players can join by going to the provided URL.

## Render Deployment Guide
1. Create an account on Render.
2. Connect your GitHub repository to Render.
3. Deploy your app by following the on-screen instructions.

## Project Structure
- `/src` - Contains all the source code.
- `/public` - Static files for the web interface.
